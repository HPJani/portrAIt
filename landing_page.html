About portrAIt

portrAIt is an AI-powered Portfolio Copilot built to act as an interactive guide through my professional and educational background, competencies, skills, projects, and experiences. It answers any and all questions related to my resume and portfolio.It was created as part of a mini-project assigned to me during my first internship at a startup called Caarya.

I utilized HTML, CSS, and Vanilla JavaScript for the basic user interface and for handling asynchronous client side DOM event logic. For the actual AI intergration, I utilisied groq console API to utilise the llama-3.3-70b-versatile model. The requisite backend utilises Node.js and Express to implement the API calls.
